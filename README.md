CannaMetrc
==========

The goal of CannaMetrc is to provide a wrapper around the Metrc REST API.

Installation
------------

You can install CannaMetrc from github with:

``` r
# install.packages("devtools")
devtools::install_github("CannaData/CannaMetrc")
```
